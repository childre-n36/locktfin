LOCKTFIN – Focus Timer App
=========================

Locktfin is a minimal, aesthetic focus timer built to help you
lock in, finish sessions, and stay consistent without distractions.

--------------------------------------------------
FEATURES
--------------------------------------------------
• Pomodoro-style focus & break timer
• Custom focus time, break time, and session count
• Session progress dots (visual progress)
• Session completion popup
• One-click full reset after session completion
• Built-in to-do list
• Floating preview window
• Smooth animations & clean UI
• Offline support (PWA-ready)

--------------------------------------------------
HOW TO USE
--------------------------------------------------
1. Launch the app
2. Set:
   - Focus duration
   - Break duration
   - Number of sessions
3. Click "Apply"
4. Press "Start" to begin
5. After each focus/break ends, click the button to continue
6. After the final session, a "SESSION OVER" popup appears
7. Close the popup to reset and start fresh

--------------------------------------------------
CONTROLS
--------------------------------------------------
Start  → Starts a new timer session  
Pause  → Pauses the current timer  
Reset  → Resets everything to a fresh session  
Preview → Opens a mini floating timer window
